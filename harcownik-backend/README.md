## Parę słów o API
To API to część backendu apikacji `Harcownik`.
<br></br>

## Jak uruchomić?
W celu uruchomienia API należy użyć komendy `uvicorn src.main:app --reload`
<br></br>

## Dokumentacja API
Dokumentacja dostępna jest po uruchomieniu API pod adresem `<adres_servera>:8000/docs`.
